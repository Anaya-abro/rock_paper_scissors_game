✊✋✌️ Rock Paper Scissors Game

A simple Python implementation of the classic Rock, Paper, Scissors game. Play against the computer in the terminal!

---

## 🎮 Features

- Play Rock, Paper, Scissors against the computer 🤖
- Random computer choice using Python's `random` module
- Instant result (Win / Lose / Tie)
- Continuous gameplay using a loop
- Beginner-friendly code

---

## 🕹️ How to Play

1. Run the Python file
2. Enter your choice:
   - rock 🪨
   - paper 📃
   - scissors ✂️
3. The computer will randomly choose its move
4. The result will be displayed:
   - YOU WIN 🎉
   - Computer wins 🤖
   - GAME TIE 🤝
